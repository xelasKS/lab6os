# lab6os
1.1 Сортировка пузырьком (Bubble Sort) — это простой алгоритм сортировки, который многократно проходит по списку, сравнивает соседние элементы и меняет их местами, если они находятся в неправильном порядке. Этот процесс повторяется до тех пор, пока список не будет отсортирован. Сложность алгоритма составляет $O(n^2)$.

1.2 Используемые структуры данных
List<T> (ArrayList): Динамический массив, обеспечивающий быстрый доступ к элементам по индексу.

Dictionary<K, V> (HashMap): Коллекция пар «ключ-значение». Сортировка пузырьком применялась к набору ключей.

Stack<T>: Коллекция, работающая по принципу LIFO («последним пришел — первым ушел»). Сортировка реализована с использованием перекладывания элементов во временный стек.
